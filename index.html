DJS

Ever been at a party and the DJ is playing all the wrong tunes? DJS is the solution and allows users to access a webpage where they can vote to select the next song. When the current song is completed the song with the most selections by users’ votes plays next. This provides a method to pick a song that majority will enjoy



CrossBorderShopping

This iOS application was developed during Hack The North 2016 with a group of three people. This app uses location based services to compare Apple product prices between USA and Canada. The price is calculated using real-time currency conversion and involves taxes and gas price to compare and find the cheapest option. The application uses REST APIs from XE Currency exchange and Google Geocoding for location.

Gas Guzzler

This web app allows the user to select a car from a database, then asks the user for their location, and a gas budget. The program uses graphing algorithms to find a path that starts from the users’ city, and uses the miles per gallon of the car selected. Then the most “exciting” path is calculated. The "exciting" factor is proportional to the population of the city.


CrosswordWeb

This web application uses gamification to record and analyze medications taken by a senior. Using New York Times crosswords the application allows the user to play only if the user enters what medicine and amount they have taken. The data is stored and then emailed to any caretaker or doctor that is taking care of the senior.
