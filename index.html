Recent Projects (Click Card For Info/Repo)

DJS

This project was created during the school year with a group of three friends and includes documentation. The application was designed to allow users to access a webpage where they can vote to select the next song to play. DJS was created through Node.js using Javascript and tested with libraries such as Selenium.

CrosswordWeb

Created a web application that records medication taken by seniors through a crossword game. The application will allows users to play only if they have entered what medicine and amount they have taken. The data is stored and emailed to the corresponding doctor.


Gas Guzzler

This web application uses public data sets to help users plan a cost efficient road trip. The project uses graphing algorithms to find a path that starts from the users’ city, and ends at the destination within the gas budget.



CrossBorderShopping

This iOS application was developed during Hack The North 2016 with a group of three people. The application provides prices between the US and Canada and compares them to find the best deal. Using REST API’s such as XE Currency and Google Geocoding the price is calculated with real-time currency conversion, and taxes.
