Dynamic DJ

A web app that uses NodeJs to host a server on a computer that plays music. Allows users’ to access a web page where they can select a song from a list made available. When the current song is completed the song with the most selections by users’ votes plays next. This provides a method to pick a song that majority will enjoy.

  • Date: October 2016 - November 2016

Gas Guzzler

This web app allows the user to select a car from a database, then asks the user for their location, and a gas budget. The program uses graphing algorithms to find a path that starts from the users’ city, and uses the miles per gallon of the car selected. Then the most “exciting” path is calculated. The "exciting" factor is proportional to the population of the city.

  • Github Repo Click Here!
  • Date: February 2016 - March 2016
  • Programming Language: Java
